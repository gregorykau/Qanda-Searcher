What's this?

While the ABC's official Q&A website provides transcripts for all episodes from 2009 onwards, they're presented in a largely static format. This project aims to provide an alternative presentation which places greater emphasis on interactivity.


Background

This project started as an assignment for a data visualization unit at Monash University. The aim was to create an interactive visualization from an existing data-set to tell a narrative. Unfortunately, no pre-processed data-sets for Q&A existed. A web-scraper was therefore written to automatically retrieve details from the online transcripts (audience voting, panelist/audience names etc.) allowing for the generation of associated statistics and visualizations.


Ongoing development goals

  1. Produce visualization for political party representation among the panel
  2. Refinement of transcripts, correcting any errors made in parsing and those present in the original ABC pages
